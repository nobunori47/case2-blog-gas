# セットアップ手順

## 1. GASへのデプロイ
1. Googleスプレッドシートを開く
2. 拡張機能 → Apps Script
3. `Code.gs` の内容を貼り付けて保存

## 2. APIキーの設定
Apps Scriptエディタで：
1. プロジェクトの設定（歯車アイコン）→ スクリプト プロパティ
2. プロパティ名: `CLAUDE_API_KEY`、値: AnthropicのAPIキー を追加

## 3. スプレッドシート構成
| 列 | 内容 |
|----|------|
| A  | キーワード |
| B  | サブキーワード |
| C  | ステータス（未生成/生成中/生成済み/エラー） |
| D  | タイトル（生成結果） |
| E  | 記事本文（HTML、生成結果） |
| F  | メタディスクリプション（生成結果） |

- 別シート名「設定」のA1セルにサンプル記事を貼り付ける（トーン参考用）

## 4. 使い方
1. スプレッドシートを再読み込みするとメニューに「記事生成」が追加される
2. A列にキーワード、B列にサブキーワードを入力
3. C列のステータスを「未生成」に設定（任意）
4. 生成したい行を選択（複数行も可）
5. 「記事生成」→「選択行の記事を生成」をクリック
